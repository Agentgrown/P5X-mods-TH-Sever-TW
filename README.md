# Persona 5 Phantom x Mods Thai sever TW
&nbsp;&nbsp;&nbsp;คือมันเป็นแปล translate autogoogle ต้องโหลด mods แปลอังกฤษมาลงก่อน แล้วถึงเอาอันนี้ไปก๊อปลงตาม folders เพราะ mods นี้ผมทำแค่แก้ config ให้เป็นแปล auto ไทย กับเอา text ภาษาไทย google มาใส่ มันจะแปลแค่ที่ต้องเจนบางอัน mods eng แปลมาแล้วผมจะไม่ยุ่ง  
&nbsp;&nbsp;&nbsp;เนื้อเรื่องหลักแปล Gimini มาอ่านรู้เรื่องแน่นอนมีแค่ ฉัน,เธอ,คะ,ค่ะ ที่ไม่ได้แก้บริบทมันอาจกจะแปลกๆหน่อย เนื่อเรื่องที่ไม่ใช่ colab แปลครบหมดแล้ว(ของ palace 4 chapter 3 v4.5.1)  
<br>
<br>  
  ---วิธีลง---  
1.) ก่อนอื่นลง mods english ของ sever Taiwan ก่อน  
<br>
2.) เข้าไปที่เกมลง mods eng ไว้ในไฟล์ตัวเกม ตัวอย่าง ผมลงไว้ใน C:\P5X\client\pc ประมาณนี้ครับ  
&nbsp;&nbsp;&nbsp;ก็ลงไปตามไฟล์เกม  
<br>
3.) ลบตัว update mods ที่ชื่อ P5XDL.dll ที่อยู่ในไฟล์ mods eng ที่อยู่ \P5X\client\pc\BepInEx\plugins  
<br>
4.) แตกไฟล์ mods ไทยที่โหลดมาแล้วว่างลงทับ files ตาม folder เลยครับ  
<br>
<br>
---เข้าเกมเล่นได้เลย---  
&nbsp;&nbsp;&nbsp;(mods จะเข้าเกมช้ากว่าปกติให้รอมันโหลดได้เลยครับ ถ้าเครื่องเก่าจะรอนานหน่อย)  
&nbsp;&nbsp;&nbsp;(มีบางครั้ง mods มันมีช่วงโหลดฉากแล้วเกม crash เพราะปัญหา memmory load (จากผมเดา) ผมเอามาแปลอีกทีทำให้ไม่มี source code เพื่อแก้หรือมันเป็นปกติของ mods ที่มันโหลดเยอะไหมไม่รู้)  
<br>
>ขอบคุณ orginal mods ที่ผมเอามาทำสาสารถโหลด mod eng ได้ที่ discord : https://discord.gg/aNwYfuby  
กับ github ที่แปลภาษาอังกฤษของคุณ https://github.com/JayJay34  
&nbsp;&nbsp;และอีกหลายๆคนที่ช่วยกันทำ  
<br>
